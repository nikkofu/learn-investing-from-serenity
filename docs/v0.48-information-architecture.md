# v0.48 信息架构（IA）与菜单重构

> 目标：把顶栏 **17 个扁平入口** 重组为**有层级、可折叠、可搜索**的导航；任意页面 ≤ 2 次点击可达。

---

## 1. 现状问题

```
[概览][方法论][趋势→产业链][个股分析][自选/收藏][热门股扫描][智能挖掘]
[回测][动量轮动][横向对比][套利雷达][盘中盯盘][策略市场][纸面交易]
[板块热力][数据同步][设置]            （+ 右侧 模型/主题/明暗）
```

- 17 项一字排开，**无分组**：用户要在「研究 / 分析 / 回测 / 交易 / 系统」之间反复扫视。
- 窄屏换行成 2–3 行，**严重挤占首屏**。
- **无搜索**：找页面靠肉眼。
- `/chart`（936 行的图表主页）**根本不在导航里**，只能从个股链接进入——可达性差。
- 无面包屑，深层页（如 `/backtest/strategy`）无层级提示。

## 3. 导航形态：侧边栏 + 精简顶栏

### 3.1 桌面布局
```
┌────────────────────────────────────────────────────────────┐
│ Topbar: [☰] Logo  ▸ 面包屑           [⌘K 搜索] [模型][主题][明暗][⚙]│
├──────────┬─────────────────────────────────────────────────┤
│ Sidebar  │                                                   │
│ (240px,  │   <PageHeader: 标题 / 副标题 / 操作区>            │
│  可折叠   │                                                   │
│  到 64px) │   <主内容区 max-w 容器>                           │
│          │                                                   │
│ 概览      │                                                   │
│ ─发现     │                                                   │
│ ─分析     │                                                   │
│ ─策略回测  │                                                   │
│ ─交易监控  │                                                   │
│ ─系统     │                                                   │
└──────────┴─────────────────────────────────────────────────┘
```
- **侧边栏**：宽 240px，可折叠为 64px（仅图标 + tooltip）；分组标题 + 项；当前项高亮（左侧 `--accent` 竖条 + `--accent-soft` 底）；折叠态记忆到 localStorage。
- **顶栏**：精简为 56px。左：折叠按钮 + Logo + 面包屑；右：全局搜索框（点击/`⌘K` 唤起命令面板）+ `ModelSelector` + `ThemeSwitcher` + `ModeToggle` + 设置齿轮。
- 图表 / 全宽页（`/chart` `/sectors`）支持**一键收起侧栏**获得最大画布。

### 3.2 移动 / 窄屏（< 768px）
- 侧栏默认隐藏，顶栏汉堡 `☰` 唤起**抽屉式侧栏**（覆盖层 + 点击遮罩关闭）。
- 顶栏搜索收为图标按钮。
- 首页栅格降为单列。

## 4. 全局搜索 / 命令面板（`Cmd/Ctrl + K`）

国际化专业工具标配（Linear / Notion / Vercel 风）。轻量自研，零新依赖：

- **页面跳转**：输入「回测 / backtest / huice」模糊匹配所有菜单项。
- **个股直达**：输入 6 位代码 → 提供「分析 `/analyze?code=` / 图表 `/chart?code=`」两个动作。
- **最近访问**：记录最近 N 个页面 / 个股（localStorage）。
- 键盘上下选择 + 回车跳转；`Esc` 关闭；命中项**新开页**（沿用 v0.47.1 `target="_blank"` 口径，导航结构性跳转除外）。

---

## 5. 面包屑

顶栏左侧显示层级，例：
- `工作台`（首页不显示或显示单级）
- `策略与回测 / 策略回测 / 单股回测`（`/backtest/strategy`）
- `分析 / K 线图表 · 601869`（`/chart?code=601869`，带个股上下文）

实现：维护 `routeMeta`（路由 → {分组, 标题, 父级}）映射表，`Breadcrumbs` 组件按当前 pathname + 关键 query 渲染。

---

## 6. 跨页链路（沿用并强化 v0.47.x 成果）
- 保留 v0.47.0/.1 已打通的研究链路（方法论→map/scanner/analyze、compare→momentum/scanner/arb 等）与 `target="_blank"` 口径。
- 新增：命令面板 + 侧栏让「主动导航」更顺；页内「跨工具」深链按钮继续保留（互补，不冲突）。
- **结构性 / 回退导航不新开页**（顶栏、侧栏、面包屑、`/chart` 返回、回测三模式切换）——延续 v0.47.1 既定例外。

---

## 7. 迁移策略（渐进、低风险）
1. 先建 `AppShell`（Sidebar + Topbar + Breadcrumbs + CommandPalette），在 `layout.tsx` 包裹，**菜单数据驱动**（单一 `navConfig.ts` 配置数组）。
2. `Nav.tsx` 退役（或保留为移动抽屉内容源）。
3. `routeMeta` / `navConfig` 单一数据源，新增页面只改配置。
4. 各页 `PageHeader` 渐进替换自写页头（见设计系统文档），不动页内逻辑。
