# v0.48 设计系统：Token 扩展与组件规范

> 原则：**在现有 `globals.css` 主题体系上只扩展不重写**。现有 5 主题 × 明暗 × 全套语义色 token 是优秀地基；本版补齐**间距 / 圆角 / 阴影 / 字体 / 字阶 / 层级 / 动效** token，并沉淀一套基础组件库，让全站视觉国际化、一致。

---

## 1. 现有 token（保留，作为色彩地基）

`globals.css` 已定义（每主题 × 明暗各一套）：
```
--bg --surface --panel --inset --nav-bg --popover-bg
--border --hover
--text --muted --faint
--accent --accent-strong --accent-soft --accent-line --accent-fg
--warn --warn-strong --warn-soft --warn-line
+ 热力图 .heat-up-3..dn-3（A 股红涨绿跌，明暗自适应）
+ .glass-card / 自定义滚动条 / 全局 cursor
```
→ **不改语义，继续作为唯一颜色来源**。新增组件一律 `var(--...)`，禁止写死 hex。

## 2. 新增 token（追加到 `:root`，全主题共享）

### 2.1 间距 Spacing（8pt 基准）
```css
--space-1: 4px;  --space-2: 8px;  --space-3: 12px; --space-4: 16px;
--space-5: 20px; --space-6: 24px; --space-8: 32px; --space-10: 40px;
--space-12: 48px; --space-16: 64px;
```

### 2.2 圆角 Radius（统一收敛，终结 md/lg/xl/2xl 混用）
```css
--radius-sm: 6px;   /* 徽章 / 小控件 */
--radius-md: 10px;  /* 按钮 / 输入 / 表格容器 */
--radius-lg: 14px;  /* 卡片 */
--radius-xl: 20px;  /* Hero / 大面板 */
--radius-full: 9999px;
```

### 2.3 阴影 / 高度 Elevation（克制；明暗各一档）
```css
--shadow-sm:  0 1px 2px rgba(0,0,0,.20);
--shadow-md:  0 4px 12px rgba(0,0,0,.28);
--shadow-lg:  0 12px 32px rgba(0,0,0,.36);
--ring-focus: 0 0 0 2px var(--bg), 0 0 0 4px var(--accent-line); /* :focus-visible */
```

### 2.4 字体 Font（**关键：补中文字栈**）
```css
--font-sans: var(--font-geist-sans), "Inter", -apple-system, BlinkMacSystemFont,
  "Segoe UI", "PingFang SC", "Noto Sans SC", "Microsoft YaHei", sans-serif;
--font-mono: var(--font-geist-mono), "SF Mono", "JetBrains Mono", monospace;
--font-num: var(--font-mono); /* 数字等宽 + font-variant-numeric: tabular-nums */
```
> 评审决策点：是否引入 `next/font` 的 **Noto Sans SC**（自托管、零外链）以保证中文渲染一致。默认建议引入（体积可子集化），否则继续系统回退。

### 2.5 字阶 Type scale

|-------|-------------------|------|

## 4. 视觉规范要点（国际化审美）

- **密度**：默认「舒适密」——卡片内边距 `--space-5`，表格行高 36–40px，区块间距 `--space-8`。
- **强调色用量**：大面积底色用 `--bg/--surface`，`--accent` 仅用于主操作 / 当前态 / 关键数据点（≤ 全屏 10%）。
- **涨跌色**：A 股红涨绿跌，来自热力图/语义 token，**不随主题强调色变**，保证财务语义稳定。
- **边框优先于阴影**：暗色界面以 1px `--border` 勾勒层次，阴影仅用于浮层（dropdown / modal / 命令面板）。
- **焦点态**：所有可交互元素 `:focus-visible` 显示 `--ring-focus`，满足 a11y。
- **动效**：仅状态反馈（hover / 展开 / 切换），`--dur` + `--ease`，禁止大幅位移 / 弹跳。
- **图标**：统一图标语言。评审决策点：引入轻量 `lucide-react`（tree-shake、与本审美契合）或维持手写内联 SVG。默认建议 `lucide-react`（稳定版）。
