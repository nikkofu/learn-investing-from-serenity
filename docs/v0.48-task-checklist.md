## 全局约束（每版必守）
- 不改回测/套利/动量/打分/复权**计算口径**；不动 `universe.ts` / `disclaimers.ts`。
- 只消费语义 token，禁写死颜色；A 股红涨绿跌不变。
- 零 / 轻新依赖（新依赖须评审 + 选发布满 7 天稳定版）。
- 三关：`tsc --noEmit` 0 error / `eslint` 改动文件 0 error（历史 warning 不计）/ `build` 通过。
- 中文 commit/tag 走消息文件；`git add` 精确路径，不 `git add .`；运行时产物（`.data/*`）与 `_*check.mts` 不提交。
