# v0.48 首页改版设计（`/` 工作台）

> 现状：`src/app/page.tsx`（90 行）是**静态介绍页**——Hero 标语 + 3 枚 CTA + 五因子卡 + 六步工作流 + 知识库 teaser。信息有价值但**不承载「今天该看什么」**，每次进站都是同一张静态页。
> 目标：改为**真实投研仪表盘（Dashboard）**——一进站即看到市场状态、我的资产、今日机会、待办告警，并能一键深入。复用现有 API，零口径改动。

---

## 1. 设计目标
- 进站 5 秒内回答：**大盘什么状态？我的自选怎样？今天有什么热门 / 机会？有没有待处理告警？**
- 保留品牌表达（Hero + 方法论入口）但下沉，让**动态数据上浮**。
- 所有模块卡可一键深入对应工具页（带上下文深链）。
- 空状态友好（无自选 / 无告警时给引导，而非空白）。

---

## 2. 布局（桌面 3 列栅格，响应式 3→2→1）

```
┌──────────────────────────────────────────────────────────────┐
│  HERO（紧凑）：Serenity 投研工作台 · 一句话定位                  │
│  [趋势→产业链拆解] [个股瓶颈点评分] [方法论/知识库]  ⌘K 搜索提示 │
├───────────────┬───────────────┬──────────────────────────────┤
│ 市场快照       │ 我的自选快照   │  今日热门（东财人气榜 Top N） │
│ 指数/涨跌家数  │ 收藏 N 只     │  代码·名称·涨跌·一键 图/分析   │
│ (KPIStat×N)   │ 涨跌分布迷你条 │                              │
├───────────────┴───────────────┼──────────────────────────────┤
│ 最近告警（alerts 最新事件）     │  快捷研究入口（工作流卡）      │
│ 命中规则 / 时间 / 查看          │  扫描 / 挖掘 / 回测 / 套利…    │
├────────────────────────────────┴──────────────────────────────┤
│ 板块热力 mini（sectors Top 涨跌板块）  │  方法论 / 五因子 / 知识库  │
└────────────────────────────────────────────────────────────────┘
```

窄屏：单列纵向堆叠，顺序 = Hero → 我的自选 → 今日热门 → 最近告警 → 快捷入口 → 市场/板块 → 方法论。

---

## 3. 模块设计（数据源均为现有接口，best-effort 容错）

| 模块 | 内容 | 数据源（现有） | 深入 |
|------|------|----------------|------|
| **Hero（紧凑）** | 定位语 + 3 CTA + `⌘K` 提示 | 静态 | map / analyze / methodology |
| **市场快照** | 主要指数 / 涨跌家数 / 情绪（KPIStat） | 复用行情源 `sources/`（沪深指数）；无则降级隐藏 | `/sectors` |
| **我的自选快照** | 收藏数、今日涨跌分布迷你条、Top 异动 | `/api/watchlist/favorites` + 实时价 | `/watchlist` |
| **今日热门** | 东财人气榜 Top 10 + 一键 图/分析 | `/api/market/hot-list`（v0.42 已建） | `/scanner?codes=...` |
| **最近告警** | 最新 N 条命中事件 + 规则 + 查看 | `/api/alerts/events` | `/alerts` |
| **快捷研究入口** | 工作流卡：扫描/挖掘/回测/套利/动量/对比 | 静态导航卡 | 各页 |
| **板块热力 mini** | Top 涨 / Top 跌板块色块 | 复用 `/sectors` 数据接口 | `/sectors` |
| **方法论 / 五因子 / 知识库** | 保留现有五因子 + 方法论 teaser（下沉） | 静态 | `/methodology` |

> 数据加载：各卡**独立 fetch + Skeleton 占位 + 失败降级**（沿用 `/analyze` 基本面面板的「独立 fetch 不阻断」范式），任一模块挂掉不影响整页。

---

## 4. 视觉与交互
- 卡片统一用新 `Card`（`--surface`+`--border`+`--radius-lg`+`--shadow-sm`），区块用 `SectionTitle`（标题 + 「查看全部 →」）。
- KPI 用 `KPIStat`：大号 mono 数字 + 涨跌箭头 + A 股红涨绿跌。
- 模块卡 hover 抬升（`--shadow-md`）+ 右上角深入箭头；点击**新开页**（沿用 v0.47.1 口径）。
- 加载用 `Skeleton`；空状态用 `EmptyState`（如「还没有收藏 → 去扫描添加」）。
- 自适应：`grid` + `auto-fit/minmax`，3→2→1 列。

---

## 5. 保留与取舍
- **保留**：3 枚主 CTA（趋势→产业链 / 个股评分 / 方法论）、五因子、知识库入口、底部免责（走 `footer`）。
- **下沉**：六步工作流 → 收进「快捷研究入口」卡或方法论页，不再占首屏。
- **新增**：市场快照 / 自选快照 / 今日热门 / 最近告警 4 个动态模块 + `⌘K` 提示。
- **不改**：任何数据口径；首页只做「读取 + 展示 + 深链」。

---

## 6. 验收点
- 有 / 无自选、有 / 无告警、热门榜可用 / 不可用 四类状态都渲染正常（实测）。
- 三档断点（>1280 / 768–1280 / <768）布局正确。
- 各模块深链带对上下文（codes / 规则）。
- 三关全绿 + 录屏（含暗/亮、至少 2 套主题切换观感）。
